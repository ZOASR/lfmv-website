## Quick start

Install it:

```bash
npm i @lastfm-viewer/[react|svelte|solid]
# or
yarn add @lastfm-viewer/[react|svelte|solid]
# or
pnpm add @lastfm-viewer/[react|svelte|solid]
```

# Get Started

```sh
npm i
# or
pnpm i
# or
yarn install
```

# Run the developement server

```sh
pnpm run dev
#or
npm run dev
```
